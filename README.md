# oas_e2e

## Overview
**oas_e2e** is an end-to-end testing framework validates API specification and ensures endpoints behave as expected.

## Features
- Easy-to-use API client based on Axios
- Supports GET, POST, PUT, PATCH, DELETE HTTP methods
- Customizable request configuration
- TypeScript support for type safety
- Extensible for various testing scenarios

## Getting Started

### Prerequisites
- Node.js (v14 or higher recommended)
- npm or yarn

### Installation
Clone the repository and install dependencies:
```sh
git clone <repository-url>
cd oas_e2e
npm install
